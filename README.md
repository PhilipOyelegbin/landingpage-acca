# ACCA Landing Page

Designed and built a landing page for ACCA (Association of Chartered Certified Accountants) using the following page as a guide for content and structure only: https://old.icanonlinetutors.com.ng/acca/

---

## Requirements

- Node v24
- PNPM v10

---

## Setup

1. Clone the repo using the command below

   ```bash
   git clone https://github.com/PhilipOyelegbin/landingpage-acca
   ```

2. Install required packages

   ```bash
   pnpm install
   ```

3. Start the website

   ```bash
   pnpm run dev
   ```

   > The wesite will be accessible the url (http://localhost:5173)
